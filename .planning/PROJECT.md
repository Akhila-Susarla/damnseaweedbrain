# DamnSeaweedBrain — BSD-Themed Portfolio

## What This Is

A fully immersive Bungou Stray Dogs-themed portfolio website for Akhila Susarla, built around the persona of Dazai Osamu. The site transforms a traditional data science portfolio into an interactive experience blending visual novel mechanics, RPG-style ability cards, and smooth scroll animations — all wrapped in a dark, witty noir aesthetic inspired by the Armed Detective Agency's world. Domain: damnseaweedbrain.com.

## Core Value

Visitors experience a memorable, game-like portfolio that showcases Akhila's data science skills and projects through the lens of the BSD universe — standing out from every other portfolio on the internet.

## Context

- **Source material:** Bungou Stray Dogs (anime/manga) — features characters named after real Japanese literary figures with supernatural abilities. Has multiple games (mobile, console) with extractable assets.
- **Character choice:** Dazai Osamu — brilliant, eccentric strategist known for dark humor and bandaged appearance. His ability "No Longer Human" nullifies other abilities. The domain "damnseaweedbrain" is a reference to how other characters address him.
- **Resume:** Located at `resources/AkhilaSusarlaResume DS GHC.docx` — Data Science focused.
- **Asset strategy:** Source BSD game assets (sprites, backgrounds, UI), fan resources, and any available official materials. Convert to web-friendly formats.
- **Sections map to BSD universe:**
  - About = Character origin story / backstory
  - Skills = Supernatural abilities with power ratings
  - Projects = Armed Detective Agency case files
  - Experience = Intel / mission dossier reports
