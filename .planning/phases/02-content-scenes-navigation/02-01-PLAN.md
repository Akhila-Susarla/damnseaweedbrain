4. Create `src/components/layout/SmoothScroll.tsx` -- a 'use client' component:
   - Uses `ReactLenis` from `lenis/react` with `root` prop
   - Options: `lerp: 0.1, duration: 1.2, smoothWheel: true`
   - Registers `gsap.registerPlugin(ScrollTrigger)` at module level
   - Syncs Lenis with GSAP: use the pattern from 02-RESEARCH.md (try ReactLenis auto-sync first, include manual fallback in comments)
   - Accepts and renders children

5. Create `src/hooks/useReducedMotion.ts` -- a 'use client' hook:
   - Reads `prefers-reduced-motion: reduce` media query on mount
   - Calls `usePortfolioStore.setReducedMotion()` to sync system preference into Zustand store
   - Listens for changes via `addEventListener('change', ...)`
   - Cleans up listener on unmount
   - Returns the `reducedMotion` boolean from store
   - Follow the exact pattern from 02-RESEARCH.md Pattern 4

6. Update `src/app/layout.tsx` -- wrap the `{children}` inside `<SmoothScroll>` component. Import SmoothScroll. The provider wraps Shell content.

7. Create `src/__tests__/reducedmotion.test.ts`:
   - Test that useReducedMotion syncs media query to store
   - Test cleanup on unmount
   - Mock window.matchMedia

1. Create `src/components/nav/NavPanel.tsx` -- a 'use client' component:

   Desktop layout (>= 768px):
   - Fixed left side rail (position: fixed, left: 0, top: 0, h-screen)
   - Width ~64-80px, bg-midnight/90 backdrop-blur
   - 6 vertical text labels in JetBrains Mono (font-mono): "Hero", "About", "Abilities", "Case Files", "Intel", "Social"
   - Labels rotated -90deg or displayed vertically, uppercase, tracking-widest, text-xs
   - Active section label highlighted in gold (text-gold), others in parchment/50
   - Thin vertical progress line next to active label that fills based on scroll progress within that section
   - z-50 to stay above content

   Mobile layout (< 768px):
   - Fixed bottom bar (position: fixed, bottom: 0, w-full)
   - Horizontal row of section labels, smaller text
   - Same active state styling (gold highlight)

   Scroll tracking:
   - Use `useGSAP` hook from `@gsap/react` to create ScrollTrigger instances for each section
   - Each ScrollTrigger: trigger `#section-id`, start 'top center', end 'bottom center'
   - onEnter/onEnterBack calls `usePortfolioStore.setCurrentSection(id)`
   - Track scroll progress via ScrollTrigger onUpdate for the progress line

   Click behavior:
   - Import `useLenis` from `lenis/react`
   - On nav label click: `lenis.scrollTo('#section-id', { offset: 0, duration: 1.2 })`
   - Each label is a `<button>` with `aria-label="Navigate to {section}"` for accessibility

   Reduced motion:
   - Read `usePortfolioStore.reducedMotion`
   - If reduced motion, skip scroll animations but keep nav functional

   Section IDs (matching what Section components will use):
   - hero, about, abilities, case-files, intel, social
