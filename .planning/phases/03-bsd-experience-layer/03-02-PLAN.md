  <action>
    1. Install R3F packages:
       ```bash
       npm install @react-three/fiber@^9.5 @react-three/drei@^10.7 three@^0.170
       npm install -D @types/three
       ```

    2. Create src/hooks/useTilt.ts:
       - Args: maxAngle (default 15)
       - Returns: { style: { transform: string; transition: string }, handlers: { onMouseMove, onMouseLeave } }
       - IMPORTANT per research anti-pattern: Do NOT use React state for mousemove. Use useRef + CSS custom properties or direct style manipulation for 60fps performance.
       - onMouseMove: calculate (clientX - rect.left) / rect.width for x (0..1), same for y. rotateX = (0.5 - y) * maxAngle * 2, rotateY = (x - 0.5) * maxAngle * 2
       - onMouseLeave: reset to rotateX=0, rotateY=0 with transition
       - Return a ref that must be attached to the element, plus event handlers
       - Include perspective in transform: "perspective(800px) rotateX(Xdeg) rotateY(Ydeg)"
       - Smooth reset transition: "transform 0.4s ease-out"
       - Active tilt transition: "transform 0.1s ease-out" (fast follow)

    3. Create src/hooks/useWebGLSupport.ts:
       - Returns boolean
       - On mount, try creating a canvas and getting webgl2 or webgl context
       - If getContext throws or returns null, return false
       - Cache result in ref (only check once)

    4. Extend src/lib/store.ts:
       - Add to PortfolioState: qualityLevel: 'high' | 'low' | 'off'; setQualityLevel: (level: 'high' | 'low' | 'off') => void
       - Default qualityLevel: 'high'

    5. Create src/components/three/FallbackProvider.tsx:
       - 'use client' component
       - Context provider that exposes: { qualityLevel, webglSupported, shouldRender3D }
       - shouldRender3D = webglSupported && qualityLevel !== 'off' && !reducedMotion
       - On mount: check useWebGLSupport(). If false, set qualityLevel to 'off' in store.
       - Export use3DQuality() hook to consume context
       - Children render regardless; 3D components check shouldRender3D before rendering Canvas

    6. Create test files for useTilt and useWebGLSupport hooks.
  </action>
